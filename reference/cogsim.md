# Simulated cognitive data

A simulated dataset based on a subset of the ADNIMERGE dataset from the
Alzheimer's Disease Neuroimaging Initiative (ADNI)

## Usage

``` r
cogsim
```

## Format

### `cogsim`

A data.table containing 1,557 rows and 4 columns:

- subid:

  Integer. Subject ID

- dementia:

  Integer. Binary indicator of dementia

- mmse:

  Integer. Analagous to MMSE_bl from ADNIMERGE

- moca:

  Integer. Analagous to MOCA_bl from ADNIMERGE
