# Print the results of a crosswalk

Print the results of a crosswalk

## Usage

``` r
# S3 method for class 'cogxwalkr.crosswalk'
print(x, ..., digits = 3L)
```

## Arguments

- x:

  An object of class "cogxwalkr.crosswalk", i.e., as returned by
  [`do_crosswalk()`](https://jrgant.github.io/cogxwalkr/reference/do_crosswalk.md)

- ...:

  Unused

- digits:

  Number of digits to print. Passed to the `digits` and `nsmall`
  arguments to [`base::format()`](https://rdrr.io/r/base/format.html).
