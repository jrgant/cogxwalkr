# Print a cogxwalkr summary

Print a cogxwalkr summary

## Usage

``` r
# S3 method for class 'summary.cogxwalkr'
print(x, ..., digits = 3L)
```

## Arguments

- x:

  An object of class "summary.cogxwalkr", i.e., as returned by
  [`summary.cogxwalkr()`](https://jrgant.github.io/cogxwalkr/reference/summary.cogxwalkr.md).

- ...:

  Unused

- digits:

  Number of digits to print. Passed to the `digits` and `nsmall`
  arguments to [`base::format()`](https://rdrr.io/r/base/format.html).
